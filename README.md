# 💬 AI-Driven Healthcare Assistant (WhatsApp-Based)

An AI-powered healthcare assistant built as a **team project**, fully integrated with **WhatsApp** to provide symptom guidance, alerts, and healthcare assistance — **without any traditional UI**.

Users interact directly through WhatsApp, making the solution highly accessible and easy to use.

---

## 🚀 Project Overview

This project leverages WhatsApp as the primary interface to:
- Understand user symptoms (non-diagnostic)
- Send smart alerts and reminders
- Provide basic healthcare guidance
- Notify caregivers when required

Designed with a focus on **real-world feasibility**, **scalability**, and **hackathon readiness**.

## ⚙️ How It Works

1. User sends a message via WhatsApp  
2. WhatsApp API receives the message  
3. n8n workflow processes the request  
4. AI model generates a response  
5. Backend stores / retrieves user data  
6. Response is sent back to the user via WhatsApp
